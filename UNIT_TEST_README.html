前端单元测试页面

概述

这是一个专门用于测试前端 JavaScript 函数的页面，目前主要用于测试数学公式渲染相关的函数。

访问方式

在浏览器中访问：http://localhost:5175/unit-test （开发服务器运行时）

或通过导航栏的 "🧪 单元测试" 链接访问。

支持的测试函数

1. renderLatex(text)

将包含 LaTeX 公式的文本转换为 HTML。

参数：

  • text (string): 包含 LaTeX 公式的文本

返回值：

  • HTML 字符串，公式部分被渲染为 KaTeX HTML

示例：

renderLatex("质量为 $m$，速度为 $v$")
// 返回: "质量为 <span>...</span>，速度为 <span>...</span>"

2. stripLatex(text)

从文本中移除 LaTeX 公式标记，替换为占位符。

参数：

  • text (string): 包含 LaTeX 公式的文本

返回值：

  • 纯文本字符串，公式被替换为 "[公式]"

示例：

stripLatex("质量 $m$ 速度 $v$")
// 返回: "质量 [公式] 速度 [公式]"

3. renderLatexWithImages(text, images)

将文本和图片组合渲染为 HTML。

参数：

  • text (string): 包含 LaTeX 公式的文本
  • images (Array): 图片对象数组，每个对象包含 dataUrl 属性

返回值：

  • HTML 字符串，包含渲染的文本和图片

使用方法

预定义测试用例

页面加载后会自动显示一系列预定义的测试用例，覆盖各种常见场景：

  • 空字符串处理
  • 简单文本
  • 行内公式 (.........)
  • 块级公式 (

    .........

    )
  • 换行符处理
  • 复杂数学公式

点击"运行所有测试"按钮执行所有测试用例，或单独点击每个测试用例的"运行"按钮。

自定义测试

在页面底部可以进行自定义测试：

  1. 选择要测试的函数
  2. 在输入框中输入参数（JSON 格式）
  3. 点击"运行自定义测试"

自定义测试示例：

// 测试 renderLatex
"牛顿第二定律：$$F = ma$$"

// 测试 stripLatex
"质量 $m$ 速度 $v$"

// 测试 renderLatexWithImages
["公式：$a^2 + b^2 = c^2$", [{"dataUrl": "data:image/png;base64,test"}]]

测试结果

每个测试用例会显示：

  • 状态: 待运行/运行中/通过/失败
  • 输入: 测试参数
  • 期望: 期望的输出结果
  • 实际: 实际的输出结果
  • 输出: HTML 渲染结果（可直接预览）
  • 错误: 如果测试失败，会显示错误信息

注意事项

  1. HTML 比较是基于字符串精确匹配，对于复杂的内容可能需要手动验证
  2. 图片相关的测试需要有效的 base64 数据 URL
  3. 所有测试都是在浏览器环境中运行的同步测试

扩展功能

如需添加更多测试函数，请在 UnitTestView.vue 中：

  1. 在 selectedFunction 的选项中添加新函数
  2. 在 runCustomTest 和 runTest 函数中添加对应的 case
  3. 在 initTestCases 中添加预定义测试用例
